# /cold-start — Initialize working posture

1) Прочитай `.memory-bank/system/memobank.yaml`.
2) Определи режим:
- greenfield-lite / standard / heavy
- brownfield-lite / standard / heavy
- fast lane
- autonomous
3) Если brownfield — сначала `/map-codebase`.
4) Если greenfield — сначала `/prd`.
5) Если задача маленькая — можно перейти в `/fast-track`.
6) Если работа продолжается — сначала `/mb-resume`.
