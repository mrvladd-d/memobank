# Deep Questioning — Техники глубокого допроса для PRD

Цель: полностью понять идею продукта до начала декомпозиции.
Принцип: лучше 5 раундов по 4 вопроса, чем 1 раунд из 3 поверхностных.

## Раунды вопросов

Задавай вопросы РАУНДАМИ по 3-5 штук. После каждого раунда:
1. Кратко суммируй что понял
2. Проверь внутренний чеклист (ниже)
3. Если есть gaps — следующий раунд
4. Если всё закрыто — предложи перейти к формированию

## Техники

| Техника | Пример |
|---------|--------|
| Конкретизация | "Ты написал X — как это выглядит на экране? Что видит пользователь?" |
| Мотивация | "Какую боль это решает? Как люди справляются сейчас без этого?" |
| Edge cases | "Что если пользователь введёт пустую строку? Что если 10 000 записей?" |
| Приоритеты | "Если бы можно было запустить только одну фичу — какую?" |
| Антифичи | "Чего точно НЕ должно быть в v1?" |
| Пользователь | "Опиши типичного юзера. Сколько ему лет? Какой опыт?" |
| UI/UX | "Как ты представляешь главный экран? Что видит пользователь первым?" |
| Конкуренция | "Есть ли аналоги? Чем твой продукт лучше?" |
| Деньги | "Как это будет монетизироваться?" |
| Масштаб | "Сколько пользователей ожидаешь через месяц? Через год?" |

## Провокация

Агент НЕ должен быть послушным стенографистом. Если видишь проблему:
- "Я вижу потенциальную проблему с X. Вот почему: ..."
- "А ты уверен что нужно Y? Обычно делают Z, потому что..."

## Внутренний чеклист (не зачитывай вслух)

Продолжай спрашивать пока не закроешь ВСЕ:

- [ ] Что за продукт? (1-2 чётких предложения)
- [ ] Для кого? (конкретная аудитория, не "все люди")
- [ ] Core Value — одна вещь, которая ОБЯЗАНА работать
- [ ] Основной user flow от входа до получения ценности
- [ ] Что в v1? Что явно НЕ в v1?
- [ ] Tech stack / ограничения / сроки
- [ ] Как выглядит успех? (метрики)
- [ ] Есть ли существующий код?
- [ ] Конкуренты / аналоги
- [ ] Монетизация (если применимо)

## Завершение

Когда все пункты закрыты:
"Я считаю, что понял идею полностью. Вот краткое резюме: [5-7 предложений].
Хочешь что-то дополнить или начинаю раскладывать по файлам?"
